# AwesomePos

> 一个 Vue 的快餐店收银系统

- element UI
- Axios
- Vue-router

## `截图`
![](https://i.loli.net/2019/07/18/5d2fd0cc46fd812645.png)

## `使用步骤`

``` bash
# 安装Vue-Cli
npm install vue-cli -g

# 下载下来

# 下载依赖
npm install

# 运行
npm run dev

# 构建
npm run build

```
